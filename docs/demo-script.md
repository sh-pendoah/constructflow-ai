# Demo Script - DocFlow 360 (docflow-360)

## Goal

Demonstrate controlled financial automation, not OCR novelty.

## Duration

2 minutes.

## Script

1. Upload three documents.
- PO, invoice, and receipt.

2. Run Multi-Way Match.
- Show decision: approved or exception.

3. Open exception details.
- Show split view with source PDF.
- Highlight bounding boxes for extracted fields.
- Show reason code and confidence.

4. Process exception.
- Choose approve override or reject.
- Show assignment/SLA metadata.

5. Show export preview.
- Display connector payload with approval metadata.
- Demonstrate export gate if approval is missing.

6. Open audit timeline.
- Show trace ID and decision chain.

## Success criteria

- Decision is explainable with source provenance.
- High-risk actions require explicit approval.
- Reviewer workload is reduced with clear reason codes.
